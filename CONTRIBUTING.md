# Contributing to Flappy's Portfolio

Terima kasih atas minat Anda untuk berkontribusi! 🎉

## 🚀 How to Contribute

### Reporting Bugs

Jika Anda menemukan bug, silakan buat issue dengan informasi:
- Deskripsi bug
- Steps to reproduce
- Expected vs actual behavior
- Screenshots (jika ada)
- Browser & OS version

### Suggesting Features

Untuk feature request, buat issue dengan:
- Deskripsi lengkap feature
- Use case / alasan mengapa feature ini berguna
- Mockup atau contoh (jika ada)

### Pull Requests

1. Fork repository
2. Create your feature branch (`git checkout -b feature/AmazingFeature`)
3. Commit your changes (`git commit -m 'Add some AmazingFeature'`)
4. Push to the branch (`git push origin feature/AmazingFeature`)
5. Open a Pull Request

### Code Style

- Gunakan TypeScript untuk type safety
- Follow existing code style
- Gunakan ESLint dan Prettier (jika tersedia)
- Buat commit messages yang jelas dan deskriptif

### Testing

Sebelum submit PR, pastikan:
- [ ] Code berjalan tanpa error (`pnpm dev`)
- [ ] Build berhasil (`pnpm build`)
- [ ] Tidak ada TypeScript errors
- [ ] UI responsive di berbagai screen sizes

## 📝 Development Workflow

1. Clone repository
2. Install dependencies: `pnpm install`
3. Create branch: `git checkout -b your-branch-name`
4. Make changes
5. Test locally: `pnpm dev`
6. Build: `pnpm build`
7. Commit & push
8. Create PR
