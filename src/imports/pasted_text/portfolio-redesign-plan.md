Perbaiki desain website portfolio agar lebih terstruktur, memiliki alur storytelling yang jelas, dan menampilkan project secara lebih profesional dan komunikatif.

Tujuan:
Mengubah portfolio dari sekadar visual showcase menjadi portfolio yang memiliki struktur kuat, menampilkan karya secara jelas, dan memperkuat personal branding.

Perubahan Utama:

1. Ganti Section:
- Ubah “Selected Work” menjadi “What I Do”
- Fokus bukan hanya menampilkan karya, tapi menjelaskan apa yang telah dibuat

2. Struktur What I Do (DIPECAH PER SECTION):

Buat section terpisah untuk setiap project, bukan dalam satu grid.

---

SECTION 1: Website Monitoring Tahfidz
- Judul: “Website Monitoring Hafalan Tahfidz”
- Sub:
  “Platform sederhana untuk membantu pencatatan dan monitoring hafalan secara digital”
- Tampilkan preview:
  - halaman dashboard / monitoring
  - gunakan tampilan menyerupai UI asli dari project (seperti web app)
- Layout:
  - kiri: teks
  - kanan: preview UI (mockup / frame browser)
- Tambahkan sedikit deskripsi fungsi:
  - pencatatan hafalan
  - tracking progress

---

SECTION 2: Landing Page Project
- Judul: “Landing Page Project”
- Sub:
  “Landing page modern untuk kebutuhan promosi jasa dan digital presence”
- Tampilkan preview:
  - hero section
  - bagian layanan
- Layout:
  - kanan teks / kiri visual (alternating layout)
- Gunakan tampilan seperti real website (full width preview atau cropped section)

---

SECTION 3: Portfolio Website
- Judul: “Portfolio Website”
- Sub:
  “Website personal untuk menampilkan identitas dan karya secara profesional”
- Tampilkan preview halaman portfolio

---

3. Hapus Section:
- Hapus “Services / What I Do” versi lama (yang berbentuk card)
- Semua diganti dengan section project di atas

---

4. Animasi PERSONAL BRANDING (SEBAGAI DIVIDER):

Ubah fungsi animasi menjadi pembatas antar section, bukan hanya dekorasi.

- Gunakan teks besar:
  “PERSONAL BRANDING”
- Style:
  - besar, bold, memenuhi lebar layar
  - gunakan outline / stroke typography
- Animasi:
  - bergerak horizontal (infinite scroll)
  - smooth dan konsisten
- Gunakan sebagai pemisah:
  Hero → divider → About → divider → What I Do → divider → CTA

- Ukuran:
  Samakan proporsi dengan referensi:
  tinggi besar, hampir seperti section penuh
  teks dominan dan kuat secara visual

---

5. Flow Baru Website:

Urutan wajib:
- Hero
- PERSONAL BRANDING divider
- About
- PERSONAL BRANDING divider
- What I Do (section 1, 2, 3)
- PERSONAL BRANDING divider
- CTA

---

6. Visual Style:
- Tetap playful, tapi lebih terkontrol
- Gunakan grid rapi untuk setiap section
- Jangan terlalu banyak elemen random
- Fokus pada keseimbangan antara ekspresi dan struktur

---

7. Typography:
- Gunakan hierarchy jelas:
  - Judul besar
  - Subheadline
  - Deskripsi kecil
- Pastikan semua readable

---

8. Interaksi:
- Hover pada preview:
  - zoom ringan
  - shadow
- Transisi antar section smooth (fade + slide)

---

Kesan Akhir:
Portfolio harus terasa seperti kombinasi antara desain editorial profesional dan kreativitas playful, dengan alur yang jelas, project yang ditampilkan secara nyata, dan animasi personal branding yang berfungsi sebagai penghubung antar section, bukan sekadar dekorasi.
